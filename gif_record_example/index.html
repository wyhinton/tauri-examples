This example demonstrates using tauri's window api to add startup arguments to Screen2Gif, allowing us to precisely capture capture our window and then edit the gif recording.

How it works:

  1. Click "Record GIF" to start the countdown
  2. The app captures the window position and size using Tauri's Window API
  3. A Node.js script launches Screen2Gif with custom arguments
  4. Screen2Gif records the specified region for the set duration
  5. Your GIF is saved to scripts/gif_output/ for editing
